## Installation

To set up this locally, follow these steps:

1. Clone the repository:
   ```bash
   https://github.com/sahiillx/Urban-Bites
   
2. Navigate to the project directory:

   ```bash
   cd Urban-Bites

3. Install NPM packages

   ```bash
   npm install

5. Start the application:

   ```bash
   npm run dev
